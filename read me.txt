폴더 연결
1. git init
2. git remote add origin https://github.com/porocol4/Created_by_me.git
(.git 폴더 생성됨)


데이터 올리기
git status			: git과 폴더의 연결상태 확인
git add .			: 변경된 파일 모두 트래킹
git commit -m "~~~"	: 변경사항 add시 commit 표시
git push origin master	: 데이터 전송 (로그인 필요)


데이터 내려받기
git pull origin master
